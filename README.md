# House-Price-Prediction

# Overview
This project focuses on predicting house prices using Machine Learning algorithms. The model is trained on housing datasets by performing data preprocessing, exploratory data analysis (EDA), feature engineering, and regression techniques to achieve accurate price predictions.

The project demonstrates the complete Machine Learning workflow from data cleaning to model evaluation.

# Features
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Handling Missing Values
- Continuous and Discrete Variable Analysis
- Regression Model Building
- Model Evaluation
- House Price Prediction
